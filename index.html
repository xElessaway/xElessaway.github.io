CTF Hub

A cleaner challenge flow for the GitHub Pages build

The old modal-heavy experience is replaced by dedicated collection and challenge routes.

Each challenge now has its own page with the prompt, image or supporting assets, password format, one submission field, and optional hint. That keeps the experience fast, linkable, and easier to maintain as the archive grows.

Answer checks remain intentionally lightweight. The browser compares normalized input against shipped hashes and stores solved progress locally so users can come back without losing momentum.
